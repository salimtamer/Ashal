3-4: Arithmetic Sequences

The Big Idea

f a Linear Function is a ramp, an Arithmetic Sequence is a staircase. Both go up or down at a constant rate, but while a function allows you to be at any point on the ramp (like x = 1.5 or x = 2.75), a sequence only lets you stand on the steps (only integers: 1, 2, 3 ,etc)

The Mechanics: \(a_n = a_1 + (n - 1)d\)

It’s just a logic puzzle:

  • \(a_n\): The value of the step you're trying to reach.
  • \(a_1\): Your starting point (the first step).
  • \(d\): The common difference, or the height of each step.
  • \((n - 1)\): The number of "jumps" you need to take to get there.

Why \((n - 1)\)? This is the part everyone messes up, so explain it like this: If you are already standing on the first step (\(a_1\)), you don't need to jump to get there. If you want to get to the 3rd step, you only need 2 jumps. That's why it's always \((n-1)\).

Move the sliders to change the first term and common difference. The staircase will grow or shrink.
