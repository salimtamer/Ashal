# Ashal beta 3.5

Ashal is an educational website  made mainly  to make learning simpler.

## Features

- Lesson Breakdown
- the video butler
- how to organize
- Student Wellbeing
- Learning strategies
  
## Purpose

Ashal was created to mainly to  help students understand complex study materials. By breaking down  simplifying. topics into structured lessons, the website makes learning more accessible, improves understanding, and supports efficient revision.

##  Languages.

- HTML
- CSS
- JavaScript (a bit of it because I'm still learning. )

## Plans for the  Future 

-  AI-powered study assistant
- PDF summarizer
- Smart search
- Progress tracking
- Interactive quizzes
- Personalized learning experience
