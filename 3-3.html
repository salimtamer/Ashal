3-3: Transforming Linear Functions

The Big Idea

Think of the parent function f(x) = x as the default settings of your phone. Transformations are just the "filters" or "customizations" you apply to change how it looks without changing what it is.

The Three Main Transformations

If we change f(x) = x to look like g(x) = a f(x) + k, we are doing these three things:

  • Vertical Shift (k): This is just sliding the graph up or down. If $k$ is positive, it slides up. If it's negative, it slides down. No change to the shape, just the position.
  • Vertical Stretch/Compression (a): This changes the slope (the steepness).If |a| > 1: The line gets steeper (stretched).If |a| < 1: The line gets flatter (compressed).
  • Reflection (a is negative): If a is negative, the graph flips across the $x$-axis. It’s like turning your camera filter upside down.
