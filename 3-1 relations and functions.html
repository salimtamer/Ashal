3-1: Relations and Functions

The Big Idea

A Relation is any pairing of inputs and outputs. A Function is a "well-behaved" relation where every input (x) maps to exactly one, predictable output (y).

The "Why"

Think of a vending machine. If you press a button and get two different snacks at random, the machine is broken. In math, functions allow us to model physics and economics because they provide one reliable answer for every input.


Note:Use the Vertical Line Test. If a vertical line hits your graph in more than one place, your "machine" is broken—it’s not a function.

Examples

1. {(1, 2), (2, 3), (3, 4)} is a function because each input has exactly one output.

2. {(1, 2), (1, 3), (2, 4)} is not a function because the input 1 maps to two different outputs (2 and 3).

3. The equation y = 2x + 1 represents a function because for every x value, there is exactly one corresponding y value.

4. The equation y^2 = x represents a relation but not a function because for some x values, there are two possible y values (e.g., if x = 4, then y can be 2 or -2).
